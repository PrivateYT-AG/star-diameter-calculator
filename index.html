Simple calculator to calculate a star's diameter.

How does the calculation work?

The calculator starts with the star's radius, measured in solar radii (R☉, 695,700 km). It converts this value into kilometers and then multiplies it by two to find the star's diameter.

Limitations

This calculator assumes that all stars are perfectly spherical. While this works well for most stars, it can introduce errors for those that are not. Some stars spin so quickly that they bulge at the equator (becoming oblate), making their diameters vary depending on where you measure. Examples include Vega, Altair, and Achernar, where this simple calculation doesn't provide accurate results.

Frequently Asked Questions

The calculator shows an error.

This can happen if:

  • The input is not a valid number (it contains letters, symbols, etc.)
  • The number is 0 or negative.
  • The number is greater than 10,000 (limited for stability purposes)
